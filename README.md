# SHSU Mathematics Salary Dashboard

Static GitHub Pages build for the SHSU Mathematics and Statistics salary dashboard.

Open `index.html` for the department overview. Open `person.html` for individual salary records.

Required hosted files:

- `index.html`
- `person.html`
- `styles.css`
- `site.js`
- `overview.js`
- `person.js`
- `data/analysis.js`
